# AdilOdeApp
This project calculates the prices of the Spotify and Youtube memberships for each member according to the specific date's inflation.
The member who pays the all membership, can send the result of the prices to other members for them to know via a redirectory to whatsapp or copying 
the informative message. Optionally it can be sent along with the iban.

**Link to Project:** https://adilode.netlify.app/
